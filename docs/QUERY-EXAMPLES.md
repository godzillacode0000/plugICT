## Tips for Better Results

1. **Be specific:** "Silver Bullet London entry" > "ICT trading"
2. **Use ICT terminology:** Use exact terms (FVG, not "gap")
3. **Include context:** Add session/time for better filtering
4. **Filter by playlist:** `--playlist "2023 ICT Mentorship"`
5. **Try multiple queries:** Semantic vs keyword may return different results
